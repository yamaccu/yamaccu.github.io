HTML div要素で図形を表示する

Published on 2021年8月21日
HTML/CSS

HTMLのdiv要素を使って、ブラウザに図形を表示させる方法です。


図形の作成方法

div要素のサイズを指定して、色を付けてあげれば図形になります。
試しに丸と四角を作ってみます。


See the Pen by やまっく (@yy87750722) on CodePen.


HTML

<div class="square"></div>
<div class="circle"></div>

CSS

.square {
  width: 50px;
  height: 50px;
  background: black;
}
.circle {
  width: 50px;
  height: 50px;
  background: black;
  border-radius: 50px;
}


複雑な形の図形

疑似要素（before / after）を使います。
疑似要素とは、HTMLの要素に対して、CSSで要素を追記できる方法です。
これを使えば、作った図形に2つまで図形の追加ができます。
（合計3つの図形を一つのDiv内に作れる）

以下のようなマークを作ってみます。

  • 時計マーク
  • 位置マーク
  • 虫眼鏡マーク
  • スイッチマーク

See the Pen by やまっく (@yy87750722) on CodePen.


■時計マーク

HTML

<div class="clock"></div>

CSS

.clock {
  display: inline-block;
  position: relative;
  width: 20px;
  height: 20px;
  border: 2px solid dimgray;
  border-radius: 20px;
}
.clock:before {
  content: "";
  position: absolute;
  width: 7px;
  height: 9px;
  border-left: 2px solid dimgray;
  border-bottom: 2px solid dimgray;
  top: 2px;
  left: 6px;
}

■位置マーク

HTML

<div class="location"></div>

CSS

.location {
  display: inline-block;
  position: relative;
  width: 20px;
  height: 20px;
  border: 2px solid dimgray;
  border-radius: 30px 30px 30px 0;
  transform: rotate(-45deg);
}
.location:before {
  content: "";
  position: absolute;
  width: 8px;
  height: 8px;
  background-color:dimgray;
  border: 2px solid dimgray;
  border-radius: 7px;
  top:4px;
  left:4px;
}

■虫眼鏡マーク

HTML

<div class="glass"></div>

CSS

.glass {
  display: inline-block;
  position: relative;
  width: 15px;
  height: 15px;
  border: 3px solid dimgray;
  border-radius: 20px;
}
glass:before {
  content: "";
  position: absolute;
  width: 4px;
  height: 14px;
  background-color: dimgray;
  border-radius: 5px;
  top: 14px;
  left: 14px;
  transform: rotate(-35deg);
}

■スイッチ

HTML

<div class="Switch"></div>

CSS

.Switch {
  position: relative;
  width: 20px;
  height: 42px;
  background: #ccc;
}
.Switch:before {
  content: '';
  position: absolute;
  width: 12px;
  height: 32px;
  left: 4px;
  top: 4px;
  background: #fff;
}
.Switch:after {
  content: '';
  position: absolute;
  width: 12px;
  height: 12px;
  left: 4px;
  top: 4px;
  background: #596;
}


spanでの図形表示

divを使うと前後に改行が入ります。
改行させたくない場合は、<span>を使用します。

その場合、cssに"display: inline-block;"を追加すると、divと同じように図形を表示させることができます。
